# NeuralInversion: Amortized Inference for Seismic Inversion

Transformer-based amortized variational inference network that maps plane-wave reflectivity `R(omega, p)` to a diagonal Gaussian approximate posterior over earth model parameters, combined with a hybrid Newton-Net pipeline for high-precision inversion.

For full mathematical detail, see `latex/amortized_inference.tex`.

## Overview

The classical approach to seismic inversion runs Levenberg-Marquardt optimisation from scratch for every observation. This module replaces the starting model with a trained neural network that provides:

1. **Instant posterior estimates** -- a single forward pass gives mean and calibrated uncertainty for all model parameters
2. **Warm start for Newton** -- the network prediction is close enough to the true model for quadratic convergence in few iterations
3. **Calibrated uncertainty** -- the predicted sigma provides a free approximate posterior without MCMC

## Architecture: SeismicInferenceNet

```
R(omega, p) complex                                      mu, log_sigma
    |                                                          ^
    v                                                          |
Stack real/imag -> (n_p, 2*nfreq)                    Linear heads (2x)
    |                                                          ^
    v                                                          |
Linear(2*nfreq, d_model) per token                    Mean pool over tokens
    |                                                          ^
    v                                                          |
+ ContinuousPositionalEncoding(p)              TransformerEncoder (pre-norm)
    |                                                          ^
    +----------------------------------------------------------+
```

Each slowness value is a token. The Transformer encoder learns cross-slowness (AVO) correlations via self-attention. Continuous positional encoding from physical `p` values (not integer indices) enables generalisation to unseen slowness grids.

| Hyperparameter | Value |
|:---|:---|
| d_model | 64 |
| n_heads | 4 |
| n_layers | 3 |
| d_ff | 128 |
| Dropout | 0.1 |
| Activation | GELU |
| Normalisation | pre-norm |

## Training

Data is generated by sampling perturbed earth models from a prior centred on a reference model, computing forward reflectivity with the GMM Riccati-sweep solver.

| Hyperparameter | Value |
|:---|:---|
| Training samples | 50,000 |
| Validation samples | 5,000 |
| Optimiser | AdamW (lr=1e-3, weight_decay=1e-4) |
| Schedule | Cosine annealing |
| Early stopping | Patience 20 epochs |
| Gradient clipping | 1.0 |
| Loss | Diagonal Gaussian NLL |

**Test set results** (4-layer ocean-crust model, 15 parameters):

| Metric | Value |
|:---|:---|
| Test NLL | -62.69 |
| Mean absolute error (log-space) | 0.0083 |
| Calibration (2-sigma) | 99.5% |

## Model Specificity

The trained network is specialised to one **model class**: a fixed layer count, parameter ordering, prior distribution, and frequency grid.

**Requires retraining:**
- Different number of layers (output dimension changes)
- Different geology (prior distribution shift)
- Different number of frequency bins (input projection width changes)

**Does NOT require retraining:**
- Different slowness grids (continuous positional encoding generalises)
- Different observations (this is the point of amortization)

## Hybrid Newton-Net Pipeline

1. **Network prediction** -- feed `R_obs` through trained network to get `(mu, sigma)`
2. **Newton-LM refinement** -- iterate from `mu` with exact Hessian until convergence

| | Hybrid | Random start |
|:---|---:|---:|
| Iterations | 10 | 28 |
| Final misfit | 2.61e-07 | 1.42e-22 |
| Final param error | 2.82e-05 | 1.06e-12 |
| Wall time (s) | 28.7 | 80.9 |

The network places the initial guess within the quadratic convergence basin of Newton's method, reducing iterations by ~3x and wall time by ~3x.

## Amortization Cost

| Stage | Time (s) |
|:---|---:|
| Data generation (50k samples) | 4,747 |
| Network training | 3,322 |
| **Total upfront** | **8,069** |

**Break-even**: 155 inversions (upfront cost / time saved per inversion).

The hybrid approach is justified when many inversions share the same model class (marine surveys, time-lapse monitoring) or when calibrated uncertainty is needed alongside the point estimate.

## Quick Start

```python
from pathlib import Path
from NeuralInversion.inference_config import load_inference_config
from NeuralInversion.inference_data import generate_training_data
from NeuralInversion.inference_train import train_inference_net
from NeuralInversion.hybrid_inversion import hybrid_invert_taup, load_trained_network

# Load config
cfg = load_inference_config(Path("configs/default_inference.yaml"))

# Generate data + train
paths = generate_training_data(cfg.data_dir, cfg.data)
model, history = train_inference_net(cfg, paths["train"], paths["val"])

# Hybrid inversion
result = hybrid_invert_taup(true_model, model, cfg.data.p_values)
```
